# Cartella Immagini

Questa cartella contiene le immagini e gli asset grafici del sito RepFinder.

## Struttura

- `logo.svg` - Logo principale del sito
- Puoi aggiungere qui altre immagini come:
  - `favicon.ico` - Icona del sito
  - `og-image.png` - Immagine per social media
  - `hero-bg.jpg` - Immagine di sfondo
  - Altri asset grafici necessari

## Utilizzo

Per utilizzare le immagini nel codice:

```jsx
// Per immagini nella cartella public/images
<img src="/images/logo.svg" alt="RepFinder Logo" />

// Per favicon
<link rel="icon" href="/images/favicon.ico" />
```

## Formati consigliati

- **SVG**: Per loghi e icone (scalabili)
- **PNG**: Per immagini con trasparenza
- **JPG**: Per foto e immagini complesse
- **WebP**: Per ottimizzazione delle prestazioni
